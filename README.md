# HelloRx - RxSwift 공부를 위한 간단한 예제 앱
![스크린샷 2024-03-06 오후 9 27 12](https://github.com/YeoSeongil/HelloRx/assets/75207506/129ba57f-2469-477b-a80b-9e7bcee028b5)

- UIKit
- RxSwift
- RxCocoa
- MVVM
- DI

## 개발환경
- Swift
- Xcode
- iOS 17
